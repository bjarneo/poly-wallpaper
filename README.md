## Usage

1. Open `index.html` in a modern web browser
2. Adjust the controls in the right panel to customize your wallpaper
3. Click "Regenerate" to create a new random pattern
4. Click "Download" to save your wallpaper

## Controls

- **Polygons**: Number of polygons to render
- **Polygon Size**: Size of each polygon
- **Depth**: Intensity of the wave animation effect
- **Animation Speed**: Speed of rotation and movement
- **Color Mode**: Choose between different coloring schemes
- **Color 1/2**: Custom colors for gradient mode
- **Background**: Background color
- **Brightness**: Overall brightness multiplier

## Browser Support

Requires a browser with WebGL support (Chrome, Firefox, Safari, Edge).
